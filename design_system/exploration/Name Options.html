Names built around the prompter
Cue, prompt, autocue, the scrolling line — the core teleprompter ideas, made ownable. Same mark and indigo identity throughout. Tell me which read right (or which to push further).
oncue
Speakers live and die "on cue." Confident, two beats, instantly understood.
oncue.ai/ on-CUE
cueline
The cue + the scrolling line in one word. Describes exactly what it shows.
cueline.app/ CUE-line
cuely
Coined from "cue," soft and modern. Short, friendly, very ownable.
cuely.com/ CUE-lee
prompta
"Prompt" with a clean coined tail. Says teleprompter without saying it.
prompta.ai/ PROMPT-uh
teleo
Abstract from "tele-." No baggage, premium, reads like a real company.
teleo.ai/ TEL-ee-oh
autoline
A nod to "autocue" + the scrolling line. Technical, on-the-nose, sturdy.
autoline.app/ AUTO-line
throughline
The thread a great talk never loses. Loved by speakers; distinctive.
throughline.app/ THROUGH-line
recite
Literally "to deliver lines aloud." One word, clear verb, brandable.
recite.ai/ re-CITE
promptr
Dropped-vowel "prompter." Developer-flavored, obviously what it is.
promptr.app/ PROMPT-er
