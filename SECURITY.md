# Security Policy

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Instead, report it privately via [GitHub Security Advisories](https://github.com/omniship-labs/eyeread.in/security/advisories/new). This lets us discuss and fix the issue before it's publicly disclosed.

Include as much detail as you can:

- A description of the vulnerability and its potential impact
- Steps to reproduce, or a proof of concept
- The affected version(s) or commit

We'll acknowledge your report within a few days and keep you updated as we work on a fix. Once resolved, we'll credit you in the advisory unless you'd prefer to stay anonymous.

## Supported Versions

Only the latest stable release is supported with security fixes.
